Asgard Recommendations app makes adding / displaying recommendations to your website quick and easy.



Step 1:

Select prefered recommendation displaying style for your page


The following code snippets are the ones which decide how recommendations should display in your page. Choose the most suitable one for your page considering the rest of the page layout and style. Then copy the relevant code snippet to your clipboard.


Style (A): Carousel

This style displays your recommendations in a horizontal carousel

<div id="bd_rec_Horizontal" data-template-type="carousel"></div>

Style (B): Grid

This style displays your recommendations in a grid

<div id="bd_rec_Grid" data-template-type="grid"></div>

Style (C): You Viewed People Also Viewed

This style displays "you viewed people also viewed" recommendations in your page

<div id="bd_rec_Horizontal" data-template-type="youViewed"></div>

Style (D): Pick Up Where You Left

This style displays "pick up where you left" recommendations in your page

<div id="bd_rec_Horizontal" data-template-type="pickUpWhereYouLeft"></div>

Style (D): Top Categories Trending

This style displays "top categories trending" recommendations in your page

<app-top-categories-rec placement="Horizontal" product-limit="10" category-limit="5"></app-top-categories-rec>


Step 2:

Adding the selected recommendation code snippet into your theme


The following steps guide you how to add the selected recommendation displying code snippets into your shopify theme.
Paste the snippet of the chosen style in required pages. The page types are mentioned below.
Paste it into the area where you want your recommendations to appear. We have suggested are few useful areas for each page type.
Once added, click on "save" to save your theme.


Home Page

Show recommendations in your Home page.
1. Go to "Themes" and open the "home-product.liquid" file. It will be under "Sections".
2. Place the chosen snippet under the div tag with the class "product grid".
